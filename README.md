DeferredValueSwap — AI-Powered Escrow Intelligent Contract for GenLayer
DeferredValueSwap is an Intelligent Contract for the GenLayer Asimov Testnet. 
It implements a full escrow-style deal lifecycle between a sender and a recipient, with:

Native GEN deposits locked in the contract
Stateful lifecycle: create → activate → finalize / cancel / expire
Optional disputes with manual or AI-based arbitration via gl._nondet
Time-based deadlines, protocol fees, blacklisting, and reputation counters

This repository is part of the GenLayer Builders program and can be used as:

a reference contract for escrow-like workflows
a starting point for a marketplace or freelancing dApp
a concrete example of GenLayer's Intelligent Contract features in practice
